## 🚀 Cómo empezar

En pocos minutos podrás consumir la API de FutManager:

1. Regístrate y obtén tu **token de autenticación**.  
2. Consulta los **endpoints disponibles** para usuarios, torneos y partidos.  
3. Integra FutManager en tu aplicación o sistema.
